Sequana: a set of NGS pipelines

This page is used as an additional set of resources for the Sequana project. Sequana is made of a Python library whose documentation is available on readthedocs. Source code of the Python library is available on github.

One of the main purpose of the Sequana project is to provide a set of NGS pipelines. Pipelines currently available are on pypi and sequana github organisation page. They have their own repository. For example the RNA-seq pipeline is available on sequana/rnaseq.

This page is also a place to provide complementary information to the main repositories.
